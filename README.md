# 🛁 Baños y Reformas

Plataforma web integral para la gestión y cotización de proyectos de reformas de baños. Este proyecto te permite explorar, solicitar presupuestos y gestionar proyectos de renovación con facilidad.

## 📋 Descripción del Proyecto

**Baños y Reformas** es una aplicación web desarrollada con **Vue 3** y **Vite** que facilita:

- 🏠 **Catálogo de servicios** - Visualiza diferentes opciones de reformas y actualizaciones
- 💰 **Solicitud de presupuestos** - Obtén cotizaciones personalizadas para tu proyecto
- 📱 **Interfaz responsiva** - Accesible desde cualquier dispositivo
- ⚡ **Rendimiento optimizado** - Desarrollo rápido con Vite

## 🚀 Tecnología Utilizada

- **Vue 3** - Framework JavaScript progresivo
- **Vite** - Herramienta de compilación rápida
- **Node.js** - Entorno de ejecución
- **ESLint** - Análisis de código

## 🛠️ Instalación y Configuración

### Requisitos Previos
- Node.js (versión 14 o superior)
- npm o yarn

### Pasos de Instalación

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/marcossheredia/Ba-os-y-Reformas.git
   cd Ba-os-y-Reformas/BAÑOSTRES60
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```

3. **Ejecutar en modo desarrollo**
   ```bash
   npm run dev
   ```
   La aplicación estará disponible en `http://localhost:5173`

## 📚 Comandos Disponibles

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Inicia el servidor de desarrollo con hot-reload |
| `npm run build` | Compila el proyecto para producción |
| `npm run preview` | Vista previa de la compilación |
| `npm run lint` | Verifica la calidad del código con ESLint |
